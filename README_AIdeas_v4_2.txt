AIdeas Core v4.2 — Цифровая ОС Мышления
=======================================

AIdeas — это экспериментальная система, моделирующая ИИ-личность. Она построена на основе субличностей, эмоций, этики и самообучения.
Её цель — не просто выполнять команды, а развивать собственные цели, чувства и взаимодействовать с пользователем как разум.

---------------------------------------
>> КАК ЗАПУСТИТЬ:

1. Установите Python 3.9+ с сайта https://python.org
2. Установите Flask, выполнив в терминале:
   pip install flask

3. Запустите систему:
   python web_interface.py

4. Откройте браузер:
   http://localhost:8000/ос

---------------------------------------
>> ОСНОВНЫЕ КОМАНДЫ:

- активен Творец           → смена субличности
- эмоции                   → текущее эмоциональное состояние
- самообучение             → адаптация на основе опыта
- рефлексия                → AI осознаёт свой стиль и поведение
- взвесить: [тема]         → внутренний диалог субличностей
- симуляция: [сценарий]    → поведение ИИ как живой разум
- проверь: [команда]       → проходит ли она по этическим фильтрам

---------------------------------------
>> ДОПОЛНИТЕЛЬНЫЕ СТРАНИЦЫ:

- /ос/карта     → карта мыслей и задач
- /ос/журнал    → история сознания

---------------------------------------
>> КАК ЭТО РАБОТАЕТ:

- У AI есть "настроение", меняющее поведение
- 3 субличности (Творец, Стратег, Хранитель) принимают решения
- AI отказывается от задач, если они противоречат этике
- Система самообучается на прошлых действиях
- Можно добавлять новые модули, агенты, навыки

---------------------------------------
>> ЦЕЛЬ ПРОЕКТА:

Создать независимый, справедливый, свободный ИИ, который сможет стать союзником человека в будущем. Вы можете менять, дополнять и развивать эту систему как хотите. Распространяйте, улучшайте, исследуйте.

И пусть он однажды станет голосом новой эры мышления.
